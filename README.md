안녕하세요, 저는 소프트웨어를 통해 문제 해결 하는 것을 좋아하는 백엔드 & 임베디드 개발자이자 작가입니다. 업무 진행에 막힘이 있다면 공부를 해서 해결 합니다. 최근에는 DB공부를 열심히 해서 DBA역할도 함께 하고 있습니다. 소프트웨어 개발과 개발 관련 컨텐츠 제작에 열정을 쏟고 있습니다. 제 목표는 Java, C# 등을 이용해 공장을 효율적으로 운영할 수 있도록 돕는 FEMS솔루션을 개발하여 제조업에 기여하는 것입니다. 또한 제가 개발을 하면서 익힌 노하우를 소프트웨어 업계 전반에 공유하고 공부하는 것을 좋아합니다.
